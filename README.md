# jokeBook

Creates a jokebook that can pull a random joke from a list. 
It allows for the addition of more jokes and a list of the jokes already on it.

## Why

> To practice the **creation of classes**, the **use of random numbers**, Git and GitHub **version control** and **the use of a menu** on a user interface.
